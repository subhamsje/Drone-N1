## 🚀 How to Run

### 1. 📋 Prerequisites
*   Python 3.10+ | Node.js v18+

### 2. 🧠 Launch the Backend
```bash
python -m venv venv && source venv/bin/activate
pip install -r requirements.txt
python main.py --demo
```

### 🎨 Launch the Frontend
```bash
cd frontend && npm install && npm run dev
```
